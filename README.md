# Tonic

:cocktail: Self meta-described collections framework

> **:construction::construction::construction: In progress ...**

## Usage

```
bin/tonic server
bin/tonic build
```

## Customization

Via `data/config.yaml`.

### Filters

Types:

- `tags`
- `select`
- `text`
- `numeric`
- `numeric_range`
- `slider`
- `slider_range`
- `boolean`

More options:

- `exclude` Exclude attributes from filters
- `labels` Customize label
